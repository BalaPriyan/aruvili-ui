# Aruvili UI: Enterprise Component Roadmap

This list outlines the planned and potential components for the Aruvili UI library to achieve full enterprise-grade coverage.

## 1. General Components
- [x] **Button**: Primary, Secondary, Danger, Outline, Ghost.
- [x] **IconButton**: Optimized for icon-only triggers.
- [x] **ButtonGroup**: Group related buttons together.
- [x] **Typography**: Standardized Headings, Text, and Code snippets.
- [x] **Layout**: Box, Flex, Stack, and Grid primitive components.
- [x] **Card**: Flexible container with Header, Content, and Footer.

## 2. Navigation
- [x] **Breadcrumbs**: Track user's location within hierarchy.
- [x] **Pagination**: Navigation through multiple pages.
- [x] **Stepper**: Multi-step process indicator.
- [ ] **Sidebar**: Collapsible navigation rail.
- [ ] **Navbar**: Top-level application header.
- [ ] **Menu**: Nested dropdowns and context menus.
- [ ] **Command Palette**: Powerful keyboard-driven search and action menu.
- [x] **Tabs**: Horizontal and Vertical tabbed navigation.

## 3. Data Entry
- [x] **Input**: Text, Email, Password, and Search variations.
- [x] **Select**: Single and multi-select dropdowns.
- [x] **Checkbox**: Multi-select options.
- [ ] **Radio Group**: Single-select options.
- [x] **Switch**: Binary toggle switch.
- [ ] **Slider**: Range selection.
- [ ] **DatePicker**: Calendar-based date and range selection.
- [ ] **ColorPicker**: Precise color selection.
- [ ] **Upload**: Drag-and-drop file uploaders.
- [ ] **Form**: High-level form controller with validation integration.
- [ ] **InputOTP**: Optimized for verification codes.

## 4. Data Display
- [x] **Avatar**: User profile image with fallbacks.
- [x] **Badge**: Numeric or status indicators.
- [ ] **Tag**: Categorization labels (closable, colorful).
- [ ] **Table**: Enterprise-grade data table (Sort, Filter, Virtualized).
- [x] **FileTree**: Recursive hierarchical folder structure.
- [ ] **Timeline**: Vertical list of chronological events.
- [ ] **Calendar**: Full-sized monthly/weekly view.
- [ ] **Accordion**: Collapsible content sections.
- [ ] **Tooltip**: Small popups on hover.
- [ ] **Popover**: Rich content floating panels.
- [ ] **Image**: Preview, Gallery, and Lazy Loading support.
- [ ] **Empty**: Standardized "No Data" states.

## 5. Feedback & Overlays
- [ ] **Alert**: Inline banners for important messages.
- [ ] **Modal / Dialog**: Center-screen focused overlays.
- [ ] **Drawer / Sheet**: Side-sliding panels.
- [ ] **Toast / Notification**: Corner-floating temporary feedback.
- [ ] **Progress**: Linear and circular progress indicators.
- [ ] **Skeleton**: Loading placeholders for cards and lists.
- [ ] **Spinner**: Indeterminate loading indicators.
- [ ] **Result**: Full-page success, error, or 404 screens.

## 6. Utilities & Misc
- [ ] **ConfigProvider**: Global theme and localization settings.
- [ ] **ScrollArea**: Custom-styled cross-browser scrollbars.
- [ ] **Separator**: Clean dividers for content grouping.
- [ ] **Resizable**: Panels for complex IDE-like layouts.
- [ ] **BackTop**: Shortcut for long scrolling pages.
- [ ] **Watermark**: Security overlay for enterprise data.

## 7. Advanced Charts (Planned)
- [ ] **Area / Line Chart**: Temporal data visualization.
- [ ] **Bar / Pie Chart**: Categorical data distribution.
- [ ] **Radar / Heatmap**: Complex multi-dimensional data.
